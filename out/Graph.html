Class: Graph

Graph(widthopt, heightopt, titleopt, dataopt)

Graph object which facilitates the creation of a streamgraph.

Constructor

new Graph(widthopt, heightopt, titleopt, dataopt)

Takes in 4 parameters from which the graph will be made upon initialisation. Default values are assigned to each parameter if none are given.
Parameters:
Name Type Attributes Default Description
width int <optional>
 600 width - Graph width in pixels
height int <optional>
 500 height - Graph height in pixels
title string <optional>
 'Title' title - Graph title
data string <optional>
 'https://raw.githubusercontent.com/AaronMorrisV/d3/master/movies.csv' data - URL of the data to be used

Example
let graph = new Graph(800,600, 'Movie Revenue');
// creates a graph object named 'graph' with the parameters;
// width:800, height: 600, title: 'Movie Revenue', data: 'https://raw.githubusercontent.com/AaronMorrisV/d3/master/movies.csv'.
